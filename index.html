What is CSS3?

CSS, also known as a cascading style sheet is a language in which we use to stlye the HTML of our page. CSS is used as a way to format a page, just like this one!

We use CSS by using 3 properties these include the :

  • Selector
  • Property
  • Value

We use the Selector to identify the HTML element we would like to target, we then use the Property to Style that element. For example, colour. We then set the Value to what we would like if it was colour we could say red.

What is Javascript?

Javascript is what us web developers call a scripting language. Im relativley new to it, although from what I know it looks like so much fun to learn!

JS is typically used in the same way any other programming language is. It consists of functions in which we then use in the HTML to include interactivity or more back end work.

Why should we use Javascript?

  • It allows us to create responsive web pages
  • It allows us to create interactive sites
  • It is starting to be used for back end web dev

Much like CSS, Javascript can be called from a standalone .js file within the HTML code. Making our code more attractive and cleaner.
